# sortingAnalysis

* 각각의 sorting 알고리즘을 처리 시간으로 비교
* 선택 정렬, 삽입 정렬, 버블 정렬, 합병 정렬, 퀵 정렬
* sorting.py를 제외한 모든 작업 최소화

## 파일 설명
